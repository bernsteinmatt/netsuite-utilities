# Introduction

**NetSuite Utilities** is a browser extension that supercharges your NetSuite workflow. Run SuiteQL queries, browse script logs, search roles instantly, and customize the interfaceâ€”all without leaving your browser.

Built with [Plasmo](https://docs.plasmo.com/), React, and TypeScript.

## What You Can Do

- **Run SuiteQL queries** directly in your browser with a full-featured SQL editor
- **Browse and filter script logs** with advanced filtering and virtual scrolling
- **Search records instantly** using a powerful command palette
- **View any record as JSON** with collapsible tree navigation
- **Load NetSuite modules** into the browser console for testing
- **Enhance the UI** with dark/light themes and interface tweaks
