# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in NetSuite Utilities.

## Tool Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Shift + K` | Open Command Search |
| `Ctrl/Cmd + Shift + U` | Open SuiteQL Editor |
| `Ctrl/Cmd + Shift + L` | Open Script Log Viewer |
| `Ctrl/Cmd + Shift + E` | Open Record Detail (JSON) |
| `Ctrl/Cmd + Shift + M` | Load NetSuite Modules |

## Within Tools

### Command Search

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate results |
| `Enter` | Open selected result |
| `Escape` | Close command search |

### SuiteQL Editor

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Enter` | Run query |
| `Ctrl/Cmd + Shift + F` | Format query |
| `Escape` | Close editor (dialog mode only) |

### Script Log Viewer

| Key | Action |
|-----|--------|
| `Escape` | Close viewer (dialog mode only) |

### Record Detail

| Key | Action |
|-----|--------|
| `Escape` | Close viewer |

## Optional: NetSuite Global Search

When enabled in extension settings:

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + K` | Focus NetSuite's global search box |

This shortcut adds a visual indicator next to NetSuite's search box. Note that this conflicts with Command Search (`Ctrl/Cmd + Shift + K`), so it's disabled by default.

## Notes

- On macOS, use `Cmd` instead of `Ctrl`
- On Windows/Linux, use `Ctrl`
- Shortcuts work on any NetSuite page where the extension is active
- **Side Panel Mode**: When a tool is open in the side panel, `Escape` does not close it (to prevent accidental closure while working)
